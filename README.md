## 📋 Sobre o Projeto

O **StyleFit** é uma landing page moderna e responsiva desenvolvida para academias e centros de fitness. O objetivo do projeto é apresentar a infraestrutura, os planos de treinamento e os diferenciais do ambiente, proporcionando uma excelente experiência de navegação com foco em conversão e usabilidade.

Este projeto foi construído para demonstrar e consolidar conhecimentos em desenvolvimento Front-end estático, priorizando uma estruturação semântica e um design atraente.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estruturação semântica e acessível do conteúdo.
- **CSS3:** Estilização, animações, transições e layout responsivo.
- **JavaScript:** Interatividade da página, validações simples e manipulação dinâmica do DOM.

## ✨ Funcionalidades

- Design 100% responsivo (adapta-se perfeitamente a smartphones, tablets e monitores ultrawide).
- Interface limpa e moderna com foco na experiência do usuário (UX).
- Seções informativas bem distribuídas (Sobre, Estrutura, Planos, Contatos).
- Navegação fluida.
